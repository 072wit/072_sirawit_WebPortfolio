สร้างหน้าโปรไฟล์ผู้ใช้แบบมืออาชีพด้วย HTML

คุณได้รับมอบหมายให้สร้างหน้าโปรไฟล์ผู้ใช้แบบง่าย ๆ แต่ครบวงจร โดยใช้ HTML5 หน้าเว็บควรแสดงข้อมูลส่วนตัวของผู้ใช้ ทักษะ ประสบการณ์ และช่องทางติดต่อ โดยเน้นการใช้แท็ก HTML เชิงความหมาย (Semantic HTML) เพื่อให้โครงสร้างชัดเจน เข้าถึงง่าย และเตรียมพร้อมสำหรับการจัดรูปแบบด้วย CSS ในอนาคต

สิ่งที่โจทย์ต้องการ:

  • ใช้แท็ก HTML5 เชิงความหมาย เช่น <header>, <main>, <section>, <footer>
  • สร้างส่วนหัว (Header) ที่มีชื่อโปรไฟล์ หรือชื่อเรื่องของหน้า (<h1>)
  • สร้างส่วน เกี่ยวกับฉัน (<section id="about">) พร้อมรูปโปรไฟล์ (<img>) และข้อมูลส่วนตัวด้วย <p>
  • สร้างส่วน ทักษะ (<section id="skills">) โดยใช้รายการ <ul> และ <li>
  • สร้างส่วน ประสบการณ์ (<section id="experience">) โดยใช้ตาราง <table> ที่มีอย่างน้อย 3 คอลัมน์ และ 2 แถวข้อมูล
  • สร้างส่วน ติดต่อ (<section id="contact">) พร้อมฟอร์ม <form> ที่มีช่องชื่อ, อีเมล และข้อความ พร้อมปุ่มส่ง
  • สร้างส่วนท้าย (<footer>) ที่มีข้อความลิขสิทธิ์หรือข้อมูลเพิ่มเติม
  • ลิงก์ไปยังไฟล์ style.css ภายนอก เพื่อเตรียมสำหรับการจัดรูปแบบ

ผลลัพธ์ที่คาดหวัง:
หน้าเว็บจะแสดงโปรไฟล์ผู้ใช้แบบครบถ้วน โดยมีส่วนหัวของหน้า (Header) ส่วนเนื้อหาหลัก (Main) ที่ประกอบด้วย เกี่ยวกับฉัน, ทักษะ, ประสบการณ์ (ตาราง) และแบบฟอร์มติดต่อ พร้อมส่วนท้าย (Footer) ด้านล่าง องค์ประกอบทั้งหมดต้องจัดเรียงอย่างเป็นระเบียบตามโครงสร้าง HTML5
