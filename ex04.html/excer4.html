สร้างบัตรแสดงสินค้า (Product Card)

ผู้ใช้ต้องสร้างโครงสร้าง HTML สำหรับบัตรแสดงสินค้าหนึ่งชิ้น โดยบัตรสินค้านี้จะประกอบด้วยรูปภาพสินค้า ชื่อสินค้า คำอธิบายสั้น ๆ ราคา และปุ่ม "เพิ่มลงตะกร้า" (Add to Cart) เป้าหมายคือการใช้แท็ก HTML เชิงความหมาย (Semantic HTML) เพื่อจัดโครงสร้างข้อมูลให้ชัดเจนและเข้าถึงได้ พร้อมสำหรับการจัดสไตล์ด้วย CSS ในอนาคต

สิ่งที่โจทย์ต้องการ:

  • ใช้แท็ก <article> เป็นคอนเทนเนอร์หลักสำหรับบัตรสินค้า
  • ภายในต้องมี <img> พร้อม src และ alt (ใช้ placeholder ได้ เช่น https://via.placeholder.com/150)
  • มีแท็กหัวเรื่อง (<h2> หรือ <h3>) สำหรับชื่อสินค้า
  • มีแท็ก <p> สำหรับคำอธิบายสั้น ๆ
  • มีส่วนแสดงราคา โดยใช้ <p> หรือ <div> และใช้ <span> หรือ <strong> เพื่อเน้นราคา
  • มีแท็ก <button> สำหรับ "เพิ่มลงตะกร้า" (Add to Cart)
  • ใช้แท็กเชิงความหมายอื่น ๆ ตามความเหมาะสมเพื่อให้โครงสร้างชัดเจน

ผลลัพธ์ที่คาดหวัง:
หน้าเว็บจะแสดงบัตรสินค้าหนึ่งใบ โดยมีรูปภาพอยู่ด้านบน ตามด้วยชื่อสินค้า คำอธิบาย ราคา (เช่น ฿599.00) และปุ่มเพิ่มลงตะกร้า โครงสร้างต้องอ่านง่าย เป็นระเบียบ และใช้แท็ก HTML อย่างถูกต้อง โดยยังไม่ต้องใส่ CSS
