สร้างหน้าโปรไฟล์ส่วนตัวง่ายๆ ด้วย HTML

คุณได้รับมอบหมายให้สร้างหน้าเว็บ HTML ที่แสดงข้อมูลโปรไฟล์ส่วนตัวของคุณ หน้าเว็บนี้ควรประกอบด้วยข้อมูลพื้นฐานเกี่ยวกับตัวคุณ รวมถึงชื่อ รูปภาพ และความสนใจ เพื่อฝึกฝนการใช้งานแท็ก HTML พื้นฐานให้ถูกต้อง

สิ่งที่โจทย์ต้องการ:

  • ต้องมีหัวข้อหลักสำหรับชื่อของคุณ โดยใช้แท็ก <h1>
  • ต้องมีรูปภาพโปรไฟล์ของคุณ โดยใช้แท็ก <img> พร้อมระบุ src และ alt attribute
  • ต้องมีส่วนสำหรับแนะนำตัวเองสั้นๆ โดยใช้แท็ก <p> อย่างน้อย 2 ย่อหน้า
  • ต้องมีรายการทักษะหรืองานอดิเรก โดยใช้ <ul> และ <li> อย่างน้อย 3 รายการ
  • ต้องมีลิงก์เว็บไซต์อย่างน้อย 2 ลิงก์ โดยใช้แท็ก <a>
  • โครงสร้าง HTML ต้องถูกต้องและปิดแท็กครบ

ผลลัพธ์ที่คาดหวัง: หน้าเว็บจะแสดงข้อมูลโปรไฟล์ส่วนตัวของคุณประกอบด้วย ชื่อ รูปภาพ คำอธิบาย 2 ย่อหน้า รายการทักษะ 3 รายการ และลิงก์เว็บไซต์ 2 ลิงก์
