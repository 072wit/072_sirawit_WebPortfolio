สร้างการ์ดโปรไฟล์ด้วย CSS

โจทย์นี้จะให้คุณสร้างการ์ดโปรไฟล์ง่ายๆ โดยใช้ HTML และ CSS เพื่อฝึกฝนการใช้งานคุณสมบัติพื้นฐานของ CSS เช่น การเปลี่ยนสี ขนาดตัวอักษร การจัดระยะห่าง การจัดกึ่งกลาง และการเพิ่มเงา

สิ่งที่โจทย์ต้องการ:

  • สร้างโครงสร้าง HTML สำหรับการ์ดโปรไฟล์ที่มีรูปภาพ ชื่อ และตำแหน่ง/คำอธิบายสั้นๆ
  • กำหนดความกว้างของการ์ดโปรไฟล์ (เช่น 300px) และเพิ่ม padding ภายใน
  • จัดกึ่งกลางรูปโปรไฟล์ภายในการ์ด
  • กำหนดขนาดตัวอักษรและสีสำหรับชื่อและคำอธิบาย
  • เพิ่มเงาให้กับการ์ดโปรไฟล์ (box-shadow)
  • กำหนดขอบมน (border-radius) ให้กับการ์ดและรูปภาพ
  • เปลี่ยนสีพื้นหลังของการ์ดให้เป็นสีขาวหรือสีอ่อนๆ

ผลลัพธ์ที่คาดหวัง: ควรเห็นการ์ดโปรไฟล์อยู่กึ่งกลางหน้าเว็บ พร้อมรูปภาพ ชื่อ และคำอธิบายที่ตกแต่งด้วย CSS อย่างสวยงาม
