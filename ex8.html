สร้างหน้าเว็บแนะนำตัวอย่างง่ายด้วย HTML

ในโจทย์นี้ คุณจะต้องสร้างไฟล์ HTML เพื่อแนะนำตัวเอง โดยใช้แท็ก HTML พื้นฐานเพื่อแสดงข้อมูลส่วนตัวของคุณ เช่น ชื่อ อายุ ความสนใจ และรูปภาพ รวมถึงการเพิ่มลิงก์ง่ายๆ เพื่อฝึกฝนการใช้งานโครงสร้าง HTML

สิ่งที่โจทย์ต้องการ:

  • สร้างหัวข้อหลัก (h1) แสดงชื่อของคุณ
  • สร้างย่อหน้า (p) แสดงอายุของคุณ
  • สร้างรายการแบบไม่เรียงลำดับ (ul) แสดงความสนใจอย่างน้อย 3 อย่าง
  • เพิ่มรูปภาพ (img) แทนตัวคุณ
  • เพิ่มลิงก์ (a) ไปยังเว็บไซต์ที่คุณชื่นชอบ

ผลลัพธ์ที่คาดหวัง: เมื่อเปิดไฟล์ HTML ในเว็บเบราว์เซอร์ คุณจะเห็นชื่อของคุณเป็นหัวข้อใหญ่ อายุในย่อหน้า รายการความสนใจ รูปภาพ และลิงก์ที่สามารถคลิกได้
