สร้างหน้าโปรไฟล์ผู้ใช้แบบง่าย

คุณได้รับมอบหมายให้สร้างหน้าเว็บสำหรับแสดงข้อมูลโปรไฟล์ผู้ใช้แบบง่ายๆ โดยใช้ HTML หน้าเว็บนี้ควรแสดงชื่อผู้ใช้ รูปภาพโปรไฟล์ และข้อมูลพื้นฐานบางอย่าง เพื่อให้ผู้เริ่มต้นได้ฝึกใช้แท็ก HTML พื้นฐาน

สิ่งที่โจทย์ต้องการ:

  • สร้างหัวข้อหลักของหน้าเว็บด้วยชื่อผู้ใช้ โดยใช้แท็ก <h1>
  • เพิ่มรูปภาพโปรไฟล์ โดยใช้แท็ก <img> พร้อมกำหนด src และ alt
  • สร้างส่วน "ข้อมูลพื้นฐาน" โดยใช้ <h2> และ <ul> หรือ <ol>
  • สร้างส่วน "ความสนใจ" โดยใช้ <h2> และ <ul> อย่างน้อย 3 รายการ

ผลลัพธ์ที่คาดหวัง: เมื่อเปิดไฟล์ HTML ในเว็บเบราว์เซอร์ คุณจะเห็นชื่อผู้ใช้เป็นหัวข้อหลัก ตามด้วยรูปภาพโปรไฟล์ และรายการข้อมูลพื้นฐานกับความสนใจต่างๆ ที่จัดรูปแบบด้วยแท็ก <ul> หรือ <ol> แสดงผลอย่างถูกต้องบนหน้าเว็บ
