สร้างหน้าโปรไฟล์ผู้ใช้แบบ Responsive

คุณได้รับมอบหมายให้สร้างหน้าโปรไฟล์ผู้ใช้แบบง่าย ๆ โดยเน้นการใช้โครงสร้าง HTML5 ที่มีความหมาย (Semantic HTML) และทำให้หน้านี้สามารถแสดงผลได้ดีบนอุปกรณ์ขนาดต่าง ๆ (Responsive Design) โดยไม่ต้องใช้เฟรมเวิร์ก CSS ใด ๆ

สิ่งที่โจทย์ต้องการ:

  • ใช้โครงสร้าง HTML5 Semantic อย่างน้อย 5ถึง7 แท็ก (เช่น <header>, <main>, <footer>, <section>, <article>, <aside>)
  • สร้างส่วนหัว (Header) ที่มีชื่อโปรไฟล์ (<h1>) และรูปภาพโปรไฟล์ (<img>) พร้อม alt attribute
  • สร้างส่วนเนื้อหาหลัก (Main Content) โดยใช้ <section> หรือ <article> และมี <p> หรือ <h2>
  • สร้างส่วนแสดงรายการทักษะ (Skills) โดยใช้ <ul> และ <li> อย่างน้อย 4 รายการ
  • สร้างส่วนช่องทางติดต่อ (Contact) โดยใช้ <aside> และมีลิงก์ <a> อย่างน้อย 3 ลิงก์
  • สร้างส่วนท้าย (Footer) ที่มีข้อความลิขสิทธิ์ (<p>)
  • เพิ่ม meta viewport เพื่อรองรับ responsive design
  • โครงสร้าง HTML ต้องจัดลำดับและเว้นวรรคให้อ่านง่าย

ผลลัพธ์ที่คาดหวัง:
หน้าเว็บจะแสดงข้อมูลโปรไฟล์ผู้ใช้ โดยมีส่วนหัวที่แสดงชื่อและรูปภาพ ส่วนเกี่ยวกับฉัน ส่วนทักษะ และส่วนช่องทางติดต่อ ซึ่งทั้งหมดสามารถปรับขนาดให้เหมาะสมกับหน้าจออุปกรณ์ต่าง ๆ ได้
