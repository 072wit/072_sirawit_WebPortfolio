จัดสไตล์หน้าเว็บเบื้องต้นด้วย CSS

ในโจทย์นี้ คุณจะได้ฝึกฝนการใช้ CSS เพื่อจัดแต่งหน้าเว็บ HTML อย่างง่าย เป้าหมายคือการทำให้ข้อความและปุ่มดูน่าสนใจและอ่านง่าย โดยใช้คุณสมบัติ CSS พื้นฐานต่างๆ เช่น การปรับสี ขนาดตัวอักษร การจัดตำแหน่ง และการเพิ่มลูกเล่นให้กับปุ่ม

สิ่งที่โจทย์ต้องการ:

  • กำหนดแบบอักษร (font-family) สำหรับทั้งหน้าเว็บ
  • จัดกึ่งกลางหัวข้อ (h1) และเปลี่ยนสีข้อความ
  • ปรับขนาดตัวอักษรของย่อหน้า (p) ให้อ่านง่าย
  • ออกแบบปุ่ม (button) ด้วยสีพื้นหลัง สีข้อความ padding และ border-radius
  • เพิ่ม margin ระหว่างองค์ประกอบต่างๆ

ผลลัพธ์ที่คาดหวัง: หน้าเว็บควรดูสะอาดตา หัวข้ออยู่กึ่งกลาง ข้อความอ่านง่าย และปุ่มมีดีไซน์สวยงามพร้อมมุมโค้งมน
