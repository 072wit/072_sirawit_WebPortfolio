สร้างการ์ดสินค้าอย่างง่าย

โจทย์นี้จะให้คุณฝึกฝนการใช้งาน CSS พื้นฐานเพื่อสร้างการ์ดสินค้าที่สวยงามและใช้งานได้จริง คุณจะต้องใช้ CSS เพื่อจัดรูปแบบองค์ประกอบต่างๆ เช่น รูปภาพ ชื่อสินค้า ราคา และปุ่ม รวมถึงการจัดวางเลย์เอาต์เบื้องต้น

สิ่งที่โจทย์ต้องการ:

  • สร้างการ์ด (.product-card) ที่มีความกว้าง 300px
  • จัดให้อยู่กึ่งกลางหน้าจอด้วย flexbox
  • เพิ่มเงา box-shadow
  • กำหนด border-radius ให้การ์ดและรูปภาพ
  • จัดรูปแบบชื่อสินค้าและราคาให้ชัดเจน
  • สร้างปุ่มเพิ่มลงตะกร้าพร้อม hover effect
  • จัดเนื้อหาให้อยู่กึ่งกลาง

ผลลัพธ์ที่คาดหวัง: เห็นการ์ดสินค้าที่มีรูปภาพ ชื่อ ราคา และปุ่ม พร้อมเงา ขอบโค้ง และเอฟเฟกต์เมื่อวางเมาส์
